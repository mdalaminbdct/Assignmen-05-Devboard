Question-1: What are the different ways to select an element in the DOM?

There are different ways of selecting elements in DOM. By ID, Class Name, Tag Name and Query Selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML shows all the HTML tags and texts, usually the full HTML code. innerTEXT shows only the texts inside the HTML tags and render the style. While textCONTENT doesn't render the Style and shows all the contents as plain text.

Question-3: What is event delegation in the DOM?

This is a way of using the Parent element by an event and find the child is clicked by event bubbling.

Question-4: What is event bubbling in the DOM?

It is the way of finding the event element. When a click happens, it searches from the root element to the child element and then go back to the root.

Question-5: How do you create, add, and remove elements using JavaScript?

Create an element using createElement(), Add element using appendChild()and Remove element using remove() methods.
